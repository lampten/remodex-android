# Roadmap

## Current Focus

The public baseline is aimed at a usable local-first Android client:

- QR pairing
- trusted reconnect
- thread list and detail
- text send
- stop / continue
- self-hostable bridge and relay flow

## Near-Term Improvements

- tighter Android polish and stability
- better image/history handling
- better self-hosting ergonomics
- more verification around multi-device behavior

## Deferred

- push notifications
- voice
- full Git/worktree UI parity
- full iOS feature parity
- hosted-service defaults

## Maintenance Posture

This roadmap is intentionally lightweight. The project is not on an aggressive public release cadence.
